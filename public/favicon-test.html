Favicon and App Icon Test

This page tests all the favicons and app icons for the Next Translator application.

Testing Instructions

  1. Check the browser tab to see if the favicon appears correctly
  2. On mobile devices, try adding this page to your home screen
  3. Verify that the app icon appears with the correct background on your home screen
  4. Open the app from your home screen to test the full-screen experience

Available Icons

16x16 Favicon Transparent background with colored logo
32x32 Favicon Transparent background with colored logo
Favicon ICO Multi-size icon (16x16, 32x32)
192x192 Icon Transparent background with colored logo
192x192 Maskable Icon colored logo on gradient background
512x512 Icon Transparent background with colored logo
512x512 Maskable Icon colored logo on gradient background
Apple Touch Icon colored logo on gradient background with rounded corners
